# Pipeline Evaluation

This directory contains modules to evaluate attribution accuracy and open-set detection effectiveness, and to create data visualization assets.

## Modules

- [metrics.py](file:///d:/Fingerprint/evaluation/metrics.py): Measures closed-set statistics (Precision, Recall, F1, Accuracy) alongside open-set specific curves (AUROC, False Acceptance Rate vs False Rejection Rate curves).
- [visualizer.py](file:///d:/Fingerprint/evaluation/visualizer.py): Generates plots for confusion matrices, styling feature importance histograms, and 2D t-SNE projections of stylometric space.
