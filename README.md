# Fingerprint: LLM Authorship Attribution & Stylometric Fingerprinting

A production-quality research project scaffold for large-scale LLM authorship attribution, stylometric fingerprint extraction, and open-set unknown model detection.

This project enables analysis of text samples to determine if they were generated by specific Large Language Models (LLMs) (closed-set classification) or by an unknown/unseen model (open-set detection), combining classical stylometrics with modern Transformer-based representations.

## Key Research Objectives

1. **Large-scale LLM Authorship Attribution**: Classify text generated by multiple known generator models (e.g., GPT-4, Claude-3, Gemini-Pro, Llama-3).
2. **Stylometric Fingerprinting**: Extract robust syntactic, lexical, structural, and semantic features representing generator-specific style.
3. **Hybrid Modeling**: Support both classical machine learning models (e.g., SVM, Random Forest, XGBoost) and deep learning models (Transformer-based classifiers).
4. **Open-set Unknown Model Detection**: Flag text generated by models not present in the training set (out-of-distribution detection).

## Getting Started

### 1. Installation
Clone the repository and install the dependencies:
```bash
pip install -r requirements.txt
```

### 2. Running the Pipeline
You can run different stages of the research pipeline using the orchestrator CLI:

```bash
# Show CLI options
python main.py --help

# Run the complete pipeline (Preprocessing -> Feature Engineering -> Model Training -> Evaluation)
python main.py --pipeline

# Run only feature extraction
python main.py --stage feature_engineering
```

## Research Workflow

1. **Generation (Optional)**: Generate synthetic dataset samples from various target LLMs (often performed in Google Colab).
2. **Preprocessing**: Normalize the text and segment it into uniform chunks or prefixes.
3. **Feature Engineering**: Compute lexical diversity, syntactic frequencies, and extract Transformer model embeddings.
4. **Model Training**: Train stylometric classifiers and fit open-set detectors on the known generators.
5. **Evaluation**: Generate performance metrics (accuracy, F1, AUROC) and visualize classification boundaries.
