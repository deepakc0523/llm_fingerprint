# Feature Engineering

This directory contains modules to extract numeric feature matrices from preprocessed text segments.

## Modules

- [stylometry.py](file:///d:/Fingerprint/feature_engineering/stylometry.py): Computes classical stylometric indicators (lexical diversity, average sentence lengths, punctuation histograms, parts-of-speech frequencies, word/character n-grams).
- [embeddings.py](file:///d:/Fingerprint/feature_engineering/embeddings.py): Encodes text chunks into dense neural vector representations using pretrained Transformer models from the Hugging Face hub.
